## go-engine

I've been interested in game engine development for quite a while, so here's my attempt at creating a simple engine in golang, using raylib.

### Features

For now, I don't want to create a complex UI, but rather abstract away the complexities of raylib, so that I can focus on the actual game dev.
Goals:
- Create an easy to use object creation API
- Create a scene management system
- Create a Physics Simulation
